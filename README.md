# BluetoothPrinter

### 基于蓝牙4.0的小票打印机功能组件封装，支持连接记忆（保存最后连接的外围设备UUID），自动连接，断开重连。
### 订单数据转换用的是git现成的代码库，具体链接：https://github.com/Halley-Wong/HLBluetoothDemo。
